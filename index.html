Step 3 · Pipeline configuration
Configure analysis modules
Select which analyses Claude will run. Each module adds depth to the report. Uncheck to skip for faster processing.
Analysis modules
Delta-beta calculation & filtering
Compute Δβ = After − Before for all 269,094 sites. Filter to |Δβ| > 0.2 threshold. Classify hypo/hyper.
Probe annotation to genes
Map significant CpG sites to gene names, genomic regions, and 7 core biological systems using embedded MSA manifest.
Biological interaction map
Generate interactive 7-system network. Score all 42 directed cross-system stress pathways. Tier classification (T1/T2/T3).
XELGEN Scoring System v1.0
Compute Longevity Score™, BAAS, APS, SIS, TESS per system, and all 6 domain composite scores (CVR, MSS, IRS, NRS, OSR, RCS).
AI narrative generation
Send results to Claude API for personalised clinical narrative, patient-facing plain-language summary, and watchlist generation.
Patient Health Story report
Compile all outputs into a complete patient-facing HTML report. Dark XELGEN theme. Ready to deliver directly to patients.
